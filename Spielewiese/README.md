##PointsAndDoors
Bewege dich mit hilfe der Tasten auf einem 10x10 Spielfeld.
Erfülle beim Spielen folgende Aufgaben:
 - Sammele ein Geldstück ein
 - Verlasse das Spielfeld durch die Tür
Mögliche bewegungen
    u      hoch
    d      runter
    r      rechts
    l      links
